  1. Run ./term-capture mylog and use the shell as normal.
    2. 
              
  2. Serve the repo dir: cd .. && python3 -m http.server 7878
    3. 
              
  3. Open /web/, click Scan, and select the mylog session.
    4. 
            

            

              Optional: vendor xterm.js at web/vendor/xterm/xterm.js and web/vendor/xterm/xterm.css
              to render ANSI properly. Without it, you’ll see a basic text fallback.
